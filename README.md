Flask SQLAlchemy demo
========================

Flask demo with SQLAlchemy integration.

## Tech Stack

- Python 3.13
- Project/Package Manager: [uv](https://docs.astral.sh/uv)
- Web Framework: Flask
- Database: SQLite
- ORM: Flask-SQLAlchemy
